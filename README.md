# Hub de Links Pessoal

Este projeto é um hub pessoal de links criado para uso próprio.

## Descrição
O site funciona como um agregador de links que redireciona
para serviços oficiais (WhatsApp, Spotify, JW Library, etc.)
utilizando protocolos de aplicativos quando disponíveis.

## Segurança
- Este projeto NÃO coleta dados
- NÃO solicita login, senha ou informações pessoais
- NÃO hospeda conteúdo de terceiros
- Apenas redireciona para sites e aplicativos oficiais

## Status
Uso pessoal / interno no momento.

## Observações
Alertas de segurança exibidos por navegadores podem ocorrer
devido ao uso de protocolos de aplicativos e redirecionamentos,
o que é comportamento esperado para este tipo de projeto.
